# Board Game v1

Initial version of the Board Game project.

## Features

- RollDice static class with a Roll() method
  - Rolls two six-sided dice
  - Prints results to the console
  - Grants a single bonus roll if double sixes are rolled

## Notes

- This version only implements dice rolling.
- No board, players, or scoring yet.
- Designed to be expanded in future versions into a full board game.

## How to Run

1. Open `BoardGame.cs` in Visual Studio or VS Code.  
2. Build and run the program.  
3. See dice rolls printed in the console.
